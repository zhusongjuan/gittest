# 概述

> 为公司年会写的小项目，主要功能为弹幕、聊天室和节目投票。前端使用Vue框架，聊天室功能基于websocket，后台由nodeJs实现。

## 使用方法

首先确保你已安装npm和nodeJs，安装方法自行google

下载项目，从命令行进入项目目录
``` bash
# install dependencies
npm install
```
npm install出错时，重新执行一次就好

``` bash
# serve with hot reload at localhost:8080
npm run dev
```

打开另一个命令行窗口，同样进入项目目录
``` bash
# start server
node server/index.js
```
